# UI/UX Pro Max Skill Guidelines

This file contains the core principles and rules for UI/UX design, which I (Antigravity) follow for this project.

## Core Principles
1. **Accessibility**: Contrast 4.5:1, Alt text, Keyboard nav.
2. **Touch & Interaction**: Min size 44x44px, 8px+ spacing.
3. **Performance**: Image optimization, lazy loading.
4. **Style Selection**: Consistency, SVG icons (no emojis).
5. **Layout**: Mobile-first, responsive.
6. **Animation**: 150-300ms, meaningful motion.
